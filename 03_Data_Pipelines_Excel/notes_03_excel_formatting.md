# Excel Essentials: Formatting

## 1. Core Concept

Formatting standardizes the visual representation of data to ensure
readability and professional presentation. It does not alter underlying
mathematical values — only how they are displayed.

## 2. Formatting Tools

**Borders**
Used to create visual hierarchy and define boundaries between distinct
datasets or summary rows (e.g., separating raw data from calculated totals).
Best practice: use sparingly to avoid visual clutter.

**Alignment**
Controls the positioning of data within cells. Standard convention:
- Text — left-aligned
- Numbers and financial figures — right-aligned so decimal places and
  magnitudes stack predictably for quick scanning

**Format Painter**
Copies all formatting properties (font, color, borders, number format)
from one cell and applies them to another. Useful for rapidly standardizing
appended data without manually recreating rules.

**Number Formats**
Dictates how a value is displayed without altering the underlying value.
Critical for differentiating between raw integers, percentages, and
currencies. Prevents calculation errors when numbers are accidentally
stored as text.

**Styles and Themes**
Global rule sets that apply consistent formatting across an entire workbook.
Ensures uniformity in reporting when multiple users are working from the
same file.

## 3. Tethered Wealth Applications

Client-facing portfolio reports and asset ledgers require professional,
mathematically unambiguous formatting. Strict alignment of AUM figures
and precise decimal controls for fractional shares prevent misreading
of financial data. Format Painter accelerates standardization when
appending new client data to existing templates.
